# 你需要做的事（手把手）

代码和 Actions 我已经写好了。你只需要在 GitHub 网页上做 **2 件事**：创建 Token，以及把它填进仓库密钥。做完后点一次「手动运行」，统计图就会自动更新。

---

## 第 0 步：先把仓库推到 GitHub（如果还没有）

个人主页仓库名必须和用户名一样：`kiki3231`。

如果你希望我代劳推送，直接在对话里回复：**帮我推送**。

也可以自己在项目目录执行：

```powershell
gh repo create kiki3231 --public --source=. --remote=origin --push
```

推送成功后，打开：https://github.com/kiki3231/kiki3231

---

## 第 1 步：创建 Personal Access Token（PAT）

1. 打开这个链接（登录你的 GitHub）：  
   https://github.com/settings/tokens?type=beta  
   （这是 **Fine-grained token** 页面；也可用经典 Token，见下方备选）

2. 点 **Generate new token**

3. 按下面填写：
   - **Token name**：随便写，例如 `profile-readme-stats`
   - **Expiration**：建议 90 天或 1 年（到期后需重建）
   - **Repository access**：选 **Only select repositories** → 勾选 `kiki3231`
   - **Permissions**：
     - Repository permissions → **Contents**：选 **Read and write**
     - Repository permissions → **Metadata**：保持 **Read-only**（默认即可）
     - Account permissions → **Profile**：选 **Read**（若列表里有）

4. 点最下方 **Generate token**

5. **立刻复制**那一长串 `github_pat_...`（只显示一次，丢了只能重建）

### 备选：经典 Token（Classic）

1. 打开：https://github.com/settings/tokens  
2. **Generate new token (classic)**  
3. Note 填 `profile-readme-stats`  
4. 勾选权限：`repo`、`read:user`  
5. Generate，复制 Token（形如 `ghp_...`）

---

## 第 2 步：把 Token 存进仓库 Secrets

1. 打开：https://github.com/kiki3231/kiki3231/settings/secrets/actions  

2. 点 **New repository secret**

3. 填写：
   - **Name**（必须一字不差）：`PAT_TOKEN`
   - **Secret**：粘贴刚才复制的 Token

4. 点 **Add secret**

---

## 第 3 步：手动跑一次 Actions（马上看到效果）

1. 打开：https://github.com/kiki3231/kiki3231/actions  

2. 左侧点 **Update README cards**

3. 右侧点 **Run workflow** → 再点绿色 **Run workflow**

4. 等它变成绿色勾（大约 1～2 分钟）

5. 回到个人主页：https://github.com/kiki3231  
   刷新后应看到更新后的 stats / 语言 / streak / trophies

之后会每天自动跑一次，不用你管。

---

## 常见问题

| 现象 | 怎么办 |
|------|--------|
| Actions 失败，提示 secret 不存在 | Secret 名字必须是 `PAT_TOKEN`，重新添加 |
| Actions 失败，403 / 权限不足 | Token 权限不够，按第 1 步重建并勾选 Contents: Read and write |
| 主页没有 README | 确认仓库名是 `kiki3231`，且 README 在仓库根目录 |
| 卡片还是占位文字 | 说明 Actions 还没成功跑完，去 Actions 页看红色报错 |

做完第 1～3 步后，把结果（成功/失败截图或报错文字）发给我，我可以帮你排查。
